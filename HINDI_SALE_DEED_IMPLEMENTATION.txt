   - HindiTextarea for address fields

   WITNESS SECTION:

5. ✅ Language Toggle Button
   - Located in form header
   - Easy to switch between English/Hindi
   - Visual indicator when Hindi is active

HOW TO TEST:
============

1. Start the server:
   npm run dev

2. Visit Sale Deed form:
   http://localhost:3000/sale-deed

3. Click the Language Toggle button (top right)

4. Go to Step 2 (Seller Details)

5. Type in the Name field:
   - Type: "Rituraj"
   - See: "रितुराज"
   
   - Type: "mera naam"
   - See: "मेरा नाम"

6. Try in Address field:
   - Type: "Delhi"
   - See: "देलही"

7. Test Buyer and Witness sections (Step 3 & 4)

WHAT WORKS:
===========
✅ Real-time English → Hindi conversion
✅ Works in Name, Relation, Address fields
✅ Numbers stay as numbers (mobile, age)
✅ Toggle on/off anytime
✅ Bilingual labels
✅ Helper text when Hindi active
✅ All seller/buyer/witness fields supported

FIELDS NOT CONVERTED (By Design):
==================================
❌ Mobile numbers - Stay as digits
❌ Age - Stay as digits
❌ ID numbers - Stay as entered
❌ PAN card - Stay as alphanumeric
❌ Email - Stay as English
❌ Dropdown selections - Stay as is
